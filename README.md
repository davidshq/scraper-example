# App Store Scraper Example

This example application was written to demonstrate (and test) the [TypeScript rewrite of app-store-scraper](https://github.com/davidshq/app-store-scraper/). The original repo for app-store-scraper is [https://github.com/facundoolano/app-store-scraper](https://github.com/facundoolano/app-store-scraper).

This code was generated by Dave Mackey using the Cursor IDE, as such the quality of the source should be suspect and it is likely to have bugs and other issues.

## Features

- Search for apps in the App Store
- Get detailed information about specific apps
- Download and store app icons and screenshots
- Store app reviews, ratings, and version history
- Track developer information
- Store all data in a SQLite database
- Support for multiple app collections and categories
- Country-specific data retrieval
- Language-specific results
- Caching capabilities for improved performance

## Database Schema

### Apps Table
Contains detailed information about apps including:
- Basic information (title, description, price, etc.)
- Ratings and reviews
- Version information
- Developer information
- Metadata (first discovered, last updated, etc.)

### Images Table
Stores information about app-related images:
- App icons
- Screenshots
- Download status and local storage
- Error tracking

### Developers Table
Contains developer information:
- Developer ID
- Website
- Country

## Setup

1. Clone the repository:
```bash
git clone <repository-url>
cd scraper-example
```

2. Install dependencies:
```bash
npm install
```

3. Create the data directory:
```bash
mkdir -p data/images
```

4. Create a `.env` file in the root directory with the following variables:
```env
# Database configuration
DB_PATH=./data/appstore.db

# App Store API configuration
COUNTRY=us
LANGUAGE=en
```

5. Build the project:
```bash
npm run build
```

## Usage

The example application demonstrates the following operations:
1. Searching for apps
2. Getting detailed app information
3. Downloading app images
4. Fetching reviews and ratings
5. Getting version history
6. Finding similar apps

To run the example:
```bash
npm start
```
